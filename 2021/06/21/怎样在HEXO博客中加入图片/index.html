1.加入图片的2种方式:

1.利用外链
  • 建议直接将图片上传到图床，然后复制链接，按照写法即可。类似的第三方免费图床推荐：Simple Free Image Hosting
  • 在需要显示的内容后面添加 标签就行了
  • 马克飞象
2.利用本地图片的相对路径

  • hexo根目录下的_config.yml 里的post_asset_folder:这个选项设置为true

  • hexo目录下用git base执行这样一句话npm install hexo-asset-image --save

  • 由于hexo3版本后对很多插件支持有问题，hexo-asset-image插件在处理data.permalink链接时出现路径错误，把年月去掉了，导致最后生成的路径为%d/xxx/xxx需要对其做兼容处理。通过判断当前版本是否等于3的版本做不同的路径分割。

  • 如果查看网页图片的源代码,发现有.io/之类的多余路径,需要修改插件的js文件,没有就不需要修改

    var version = String(hexo.version).split(‘.’);
