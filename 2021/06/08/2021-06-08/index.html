    3.tomcat编码问题

    1.控制台乱码

    tomcat根目录bin/conf/logging.properties,里的java.util.logging.ConsoleHandler.encoding = 设置为GBK

    2.页面乱码,

    1.修改server.xml,添加一个URIEncoding=”UTF-8”,如下:

    2.添加init-parm,在如下标签中添加

    default org.apache.catalina.servlets.DefaultServlet debug 0 listings false 1 fileEncoding UTF-8 //添加utf编码 3.配置tomcat/bin/catalina.bat中的

    set “JAVA_OPTS=%JAVA_OPTS% %JSSE_OPTS%”

    字符串里添加-Dfile.encoding=UTF8 -Dsun.jnu.encoding=UTF8

    如下:

    set “JAVA_OPTS=%JAVA_OPTS% %JSSE_OPTS% -Dfile.encoding=UTF8 -Dsun.jnu.encoding=UTF8”

    4.浏览器缓存清除,防止缓存问题,导致修改配置不生效.

    4.html页面编码

    在html页面head头部加入<meta content=”text/html”; charset=”utf-8”>.
